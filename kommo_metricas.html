Como obter o token
  • 1Acesse seu Kommo → Configurações → Integrações → Criar integração
  • 2Dê um nome qualquer (ex: Dashboard) e salve
  • 3Vá na aba "Chaves e escopos" e copie o Long-term access token
  • 4Informe o subdomínio (ex: revendeletro se a URL for revendeletro.kommo.com)
    • ⚡ URL do Proxy (obrigatório para funcionar)
    Para contornar o bloqueio CORS do navegador, é necessário um proxy.
    Crie o seu gratuitamente em workers.cloudflare.com usando o arquivo
    cloudflare-worker-proxy.js que acompanha este dashboard.
    Depois cole a URL do Worker abaixo (ex: https://meu-proxy.USUARIO.workers.dev)
